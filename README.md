🐍 Python Mastery: 0 to Advanced

Welcome to my Python Mastery Challenge! 🚀

I have taken on a challenge to learn Python from **Day 1** to **Advanced** level. I will be coding every single day without breaking the chain until I reach an advanced stage. This repository serves as my daily log, documentation, and progress tracker.

🎯 Goal
To build a deep, practical understanding of Python by starting from the absolute basics and progressively moving to advanced concepts like Object-Oriented Programming (OOP), Decorators, Generators, APIs, Data Science libraries, and Automation.

📅 Daily Plan
- Daily Commitment:Code every day.
- Structure:Each day will have its own folder (e.g., `Day_01`, `Day_02`).
- Content:Each folder will contain:
    - A Python script (`.py`) related to the day's topic.
    - A `notes.md` (optional) with key learnings.

🗺️ Learning Path (Tentative)
- Phase 1: Basics (Days 1-15)
    - Variables, Data Types, Strings
    - Lists, Tuples, Dictionaries, Sets
    - Conditionals (if/else) and Loops (for/while)
    - Functions and Scope
- Phase 2: Intermediate (Days 16-40)
    - File Handling (read/write files)
    - Error Handling (try/except)
    - Object-Oriented Programming (Classes, Inheritance)
    - Modules and Packages
    - List Comprehensions and Lambda Functions
- Phase 3: Advanced (Days 41+)
    - Decorators and Generators
    - Working with APIs (Requests library)
    - Database Interaction (SQLite/SQLAlchemy)
    - Testing (Unit tests)
    - Introduction to Data Science (Pandas, NumPy) or Web Development (Flask/Django)
